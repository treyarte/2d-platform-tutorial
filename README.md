# Lava Playground

## Purpose:
The purpose of this project was for me to create a playground of game mechanics for testing so they can be added to other games. The idea I had was similar to how [Shigeru Miyamoto](https://en.wikipedia.org/wiki/Shigeru_Miyamoto) used to design the movement/mechanics of Mario in Mario 64. I wanted a basic level to test stuff out for future games.

## About:
This Unity project is a 2D platformer designed as a testing ground for various mechanics used in my future platforming games. The tools and features developed here are created with a focus on reusability. During this project, I gained valuable experience implementing concepts like the observer pattern, which was helpful in handling player deaths and stage resets.
